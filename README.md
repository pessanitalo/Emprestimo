# Projeto Clini-Corp
Projeto em desenvolvimento, objetivo é criar um sistema de empréstimos. Onde o cliente possa simular um empréstimo e caso ele deseje fazer a contratação do mesmo.

## Tecnologias Utilizada
## Backend
- C#
- .Net 7
- ASP.NET WebApi   
- Padrão Repositório
## Frontend
- Html-Css
- Angular V13
- Ngx-Mask
- Ngx-Bootstrap
- Ngx-Toastr
